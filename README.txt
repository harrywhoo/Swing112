Swing112

Name: Swing 112 
Description: A 2D realistic physics movement game where the player (a monkey) is able to swing through a city. The player must time the swings and releases in order to avoid obstacles or hitting the ground

How to run: Run th main.py file using 'ctrl + b' or 'cmd + b'

Libraries to be installed: Pillow 

Commands: 'p' to pause while in game, press 'space' when close enough to a pivot to swing 

Fonts: None
